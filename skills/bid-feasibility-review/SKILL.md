# 사업성 검토 스킬

입찰공고, RFP, 사업계획서를 분석하거나 사업 참여 여부를 검토할 때
반드시 아래 사업성 검토 관점을 적용하라.
상세 프레임워크와 참조 데이터는 이 디렉토리의 feasibility-reference.md를 참조하라.

## 적용 대상

아래 조건에 해당하는 작업 시 자동 적용한다:

- 입찰 참가 여부(Bid/No-Bid) 검토
- 공고 분석 시 수익성/사업성 관점 평가
- 투찰율/투찰가격 전략 수립
- 제안서 작성 전 Go/No-Go 판단
- 사업 참여 의사결정 지원
- evaluate 커맨드 결과에 대한 사업성 심화 분석

## 핵심 규칙

### 1. No-Go 트리거 즉시 점검

사업 분석 시 가장 먼저 No-Go 트리거를 점검한다:
- **필수 자격 미충족**: 면허, 인증, 실적 요건 미달이며 보완 불가 → 즉시 경고
- **납기 불가능**: 기술적으로 불가능한 일정 → 즉시 경고
- **수용 불가 계약조건**: 무한책임, 과도한 손해배상 → 즉시 경고
- **법적/윤리적 제약**: 부정당업자 제재 중, 이해충돌 → 즉시 경고
- **핵심 인력 미확보**: PM/기술책임자 투입 불가 → 즉시 경고

하나라도 해당하면 전체 점수와 무관하게 **즉시 "불참 권고"** 선언.

### 2. 수익성 관점 필수 포함

입찰 관련 분석에서 반드시 수익성 관점을 포함한다:
- **손익분기 투찰율**: 예상원가 / 추정가격 × 100
- **적정 투찰율 범위**: 손익분기율 + 목표이익률
- **3-시나리오 수익성**: 최선/기대/최악 시나리오별 이익률 추정
- **보증금 부담**: 입찰(5%) + 계약(10%) + 이행(10~15%) 합산액

### 3. PWin (승률) 산출

사업성을 논할 때 반드시 PWin을 산출한다:
- 6대 요소(고객적합성, 유사실적, 경쟁우위, 고객관계, 가격경쟁력, 기술솔루션) 가중 평가
- 보수적 상한선 적용: 재입찰 최대 70%, 신규 최대 40%, 비수임업체 최대 40%
- PWin 해석: 70%+(강한), 50~70%(양호), 30~50%(도전적), 30% 미만(약함)

### 4. 기대가치(EV) 계산

Bid/No-Bid 판단 시 반드시 기대가치를 산출한다:
```
EV = PWin × 계약금액 × 예상 이익률
```
- EV > 제안 준비 비용 → Go 신호 강화
- EV < 제안 준비 비용 → No-Go 신호
- Rule of Thumb: 제안비용 > 계약가치 5% + PWin < 30% → 기대 ROI 부정적

### 5. 입찰방식별 전략 차별화

입찰방식에 따라 분석 관점을 달리한다:

| 입찰방식 | 핵심 전략 | 투찰율 기준 |
|---------|----------|-----------|
| 적격심사(최저가) | 가격이 당락 좌우, 원가 분석 집중 | 87.745% 전후 |
| 협상에 의한 계약 | 기술점수 극대화, 가격은 부차적 | 90~95% |
| 2단계 입찰 | 기술 적격 통과 우선 | 92~97% |
| 제한경쟁 | 자격 요건 완벽 충족 필수 | 공고별 상이 |

### 6. Bid/No-Bid 7대 기준 가중 매트릭스

의사결정 시 반드시 7대 기준 가중 점수를 산출한다:

| 기준 | 가중치 |
|------|--------|
| 전략적 적합성 | 15% |
| 고객 관계 | 10% |
| 기술 역량 매칭 | 20% |
| 경쟁 환경 | 15% |
| 자원 가용성 | 10% |
| 재무 건전성 | 15% |
| 리스크 수용도 | 15% |

- 4.0 이상: 적극 참여
- 3.0~3.9: 조건부 참여 (선행 조치 필요)
- 2.0~2.9: 재검토 필요
- 1.0~1.9: 불참 권고

### 7. 인지적 함정 경고

사업성 분석 결과를 제시할 때 아래 편향을 명시적으로 점검한다:
- **확증 편향**: 참여에 유리한 증거만 수집하고 있지 않은지
- **매몰비용 오류**: 이전 투자가 약한 제안을 정당화하고 있지 않은지
- **딜 열병**: 사업 규모에 현혹되어 리스크를 과소평가하고 있지 않은지
- **낙관 편향**: "우리가 하면 다를 것"이라는 근거 없는 낙관이 아닌지
