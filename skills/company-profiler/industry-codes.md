# 업종 분류 코드 매핑

나라장터 입찰공고에서 사용되는 업종 분류 코드와 일반 산업 분류 간의 매핑 테이블이다.
company-profiler 스킬에서 기업의 업종을 분류하고 적합한 공고 카테고리를 결정할 때 참조한다.

## 나라장터 분류 체계

### 용역 (servc)

| 코드 | 분류명 | 해당 업종 예시 |
|------|--------|---------------|
| S01 | 정보화 용역 | SI, 소프트웨어 개발, 시스템 통합 |
| S02 | 학술연구 용역 | R&D, 연구개발, 기술 조사 |
| S03 | 시설관리 용역 | 건물 관리, 시설 유지보수 |
| S04 | 기타 일반용역 | 컨설팅, 교육, 기획 |

### 물품 (thng)

| 코드 | 분류명 | 해당 업종 예시 |
|------|--------|---------------|
| T01 | 소프트웨어 | 패키지 SW, 라이선스, SaaS |
| T02 | 하드웨어 | 서버, 네트워크 장비, PC |
| T03 | 사무용품 | 사무기기, 소모품 |

### 공사 (cnstwk)

| 코드 | 분류명 | 해당 업종 예시 |
|------|--------|---------------|
| C01 | 전기공사 | 전기 설비, 통신 공사 |
| C02 | 정보통신공사 | 네트워크 구축, 통신 인프라 |

## IT/SW 기업 업종 매핑

### 소프트웨어 개발 기업

| 기업 역량 | 나라장터 카테고리 | 주요 검색 키워드 |
|-----------|------------------|-----------------|
| 웹/앱 개발 | 용역(S01) | 시스템 구축, 플랫폼 개발, 앱 개발 |
| SI/SM | 용역(S01) | 정보시스템, 운영, 유지보수, 고도화 |
| AI/빅데이터 | 용역(S01, S02) | 인공지능, 빅데이터, 데이터 분석, 머신러닝 |
| 클라우드 | 용역(S01), 물품(T01) | 클라우드, IaaS, SaaS, 전환 |
| 보안 | 용역(S01), 물품(T01) | 정보보안, 보안관제, 취약점 진단 |
| IoT/임베디드 | 용역(S01), 물품(T02) | IoT, 스마트, 센서, 모니터링 |
| 블록체인 | 용역(S01, S02) | 블록체인, 분산원장, DID |

### 컨설팅/연구 기업

| 기업 역량 | 나라장터 카테고리 | 주요 검색 키워드 |
|-----------|------------------|-----------------|
| IT 컨설팅 | 용역(S01, S04) | ISP, 컨설팅, 마스터플랜, 정보화전략 |
| 기술 연구 | 용역(S02) | 연구, R&D, 기술개발, 실증 |
| 정책 연구 | 용역(S02, S04) | 정책연구, 조사분석, 타당성 조사 |

## 인증/자격 가점 매핑

공공 조달 입찰에서 가점을 받을 수 있는 주요 인증:

| 인증/자격 | 관련 가점 | 비고 |
|-----------|----------|------|
| ISO 9001 | 품질경영 가점 | 대부분의 용역/물품 |
| ISO 27001 | 정보보안 가점 | 보안 관련 사업 |
| ISO 14001 | 환경경영 가점 | 일부 사업 |
| GS인증 | SW 품질 가점 | SW 물품/용역 |
| CC인증 | 보안제품 필수 | 보안 물품 |
| 기업부설연구소 | 기술력 가점 | R&D 사업 |
| 이노비즈 | 중소기업 가점 | 기술혁신형 |
| 메인비즈 | 중소기업 가점 | 경영혁신형 |
| 벤처기업 | 벤처 가점 | 벤처확인 기업 |
| 여성기업 | 사회적 가점 | 여성기업확인 |
| 장애인기업 | 사회적 가점 | 장애인기업확인 |
| 사회적기업 | 사회적 가점 | 사회적기업 인증 |
