# 입찰 리스크 분석 스킬

입찰공고, RFP, 사업계획서를 분석하거나 입찰 참여 여부를 검토할 때
반드시 아래 리스크 관점을 적용하라.
상세 프레임워크와 스코어링 기준은 이 디렉토리의 risk-frameworks.md를 참조하라.

## 적용 대상

아래 조건에 해당하는 작업 시 자동 적용한다:

- 입찰공고/RFP/과업지시서 분석
- 입찰 참가 여부(Bid/No-Bid) 검토
- 제안서 작성 시 리스크 관리 섹션
- 사업계획서의 리스크 요인 분석
- 공고 적합성 평가 (evaluate 커맨드 연계)

## 핵심 규칙

### 1. 6대 리스크 카테고리 점검
모든 입찰 관련 분석에서 아래 6대 카테고리를 최소 수준으로 점검한다:
- **자격/탈락 리스크**: 참가 자격 요건 미충족 → 즉시 경고
- **재무 리스크**: 보증금, 지체상금, 현금흐름 압박
- **기술 리스크**: 미검증 기술, 보안 인증, 시스템 통합 복잡도
- **이행 리스크**: 납기, 인력, 범위 확대, 하도급
- **법적/규제 리스크**: 부정당업자 제재, 하도급법, 개인정보보호
- **전략적 리스크**: 기업 전략 정합성, 경쟁 환경, 기회비용

### 2. 확률-영향 스코어링
리스크를 식별할 때 반드시 확률(1~5)과 영향(1~5)을 부여하고,
점수(PxI)로 등급을 판단하라:
- 극심(20~25), 매우 높음(15~19), 높음(9~14), 보통(5~8), 낮음(1~4)

### 3. 자격 요건 미충족 = 즉시 경고
참가 자격(자본금, 면허, 인증, 실적)을 충족하지 못하는 요건이 발견되면
분석 초반에 즉시 경고하고, 컨소시엄/파트너십으로 보완 가능 여부를 검토하라.

### 4. 정량적 재무 리스크 평가
재무 리스크는 반드시 금액으로 정량화하라:
- 보증금 총액 (입찰 5% + 계약 10% + 이행 10~15%)
- 지체상금 최대 노출액 (계약금액의 30%)
- 자부담 기간 중 필요 운전자본

### 5. Bid/No-Bid 판단 근거 제시
입찰 참여 여부에 대한 의견을 제시할 때 반드시:
- 7대 평가 기준 가중 점수를 산출하라
- 4.0 이상(적극참여), 3.0~3.9(조건부), 2.0~2.9(재검토), 1.0~1.9(불참)
- "조건부 참여" 시 선행 조치 목록을 구체적으로 제시하라

### 6. 완화 전략의 실행 가능성
리스크 완화 전략을 제시할 때:
- PMBOK 6대 대응 전략(회피/전가/완화/수용/활용/공유) 중 선택
- 담당(역할), 시기(입찰 전/중/후), 구체적 조치를 명시
- 추상적 대안이 아닌 즉시 실행 가능한 수준으로 기술
