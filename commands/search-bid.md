# 공고 통합 검색

$ARGUMENTS 에 지정된 키워드로 공공 조달/지원사업 공고를 검색하라.

## 인자 파싱 및 키워드 확보

다음 순서로 검색 키워드를 확보한다:

1. **$ARGUMENTS에서 확인**: 키워드가 직접 지정되었으면 사용한다
2. **이전 결과 활용**: 인자가 없으면 현재 대화에서 이전 `analyze` 결과의 키워드를 사용한다
3. **사용자에게 요청**: 키워드를 확보하지 못하면 대화형으로 요청한다:
   "검색할 키워드를 입력해주세요. (예: AI 솔루션, 빅데이터 분석, 클라우드 구축)
   여러 키워드는 쉼표로 구분하거나, 기업 문서를 먼저 분석하려면 `/proposal-specialist:analyze` 를 실행해주세요."
   - 사용자가 키워드를 입력하면 해당 키워드로 계속 진행한다

## 검색 절차

1. 위 절차로 확보한 키워드를 기반으로 검색한다
2. bid-searcher 에이전트를 호출하여 병렬 검색을 위임한다
3. 결과를 통합하여 적합도 순으로 정렬한다

검색 결과에서 입찰 마감이 지난 공고는 자동으로 제외된다.
마감 3일 이내 공고는 `[긴급]` 태그가 표시된다.

## 검색 대상 플랫폼

MCP Tool을 사용하여 다음 플랫폼을 병렬 검색하라:

| MCP Tool | 플랫폼 | 용도 |
|---|---|---|
| `g2b_search_bids` (category: servc) | 나라장터 | 용역 입찰공고 |
| `g2b_search_bids` (category: thng) | 나라장터 | 물품 입찰공고 |
| `g2b_search_pre_specs` | 나라장터 | 사전규격 (향후 공고 예정) |
| `kstartup_search_announcements` | K-스타트업 | 지원사업 공고 |
| `mss_search_announcements` | 중소벤처기업부 | 사업공고 |
| `msit_search_announcements` | 과학기술정보통신부 | 사업공고 |

## 출력 형식

마감 제외 건수와 적합도 등급을 포함하여 적합도 순으로 정렬하고, 각 공고에 대해:
- 공고명, 기관명, 마감일 (D-{남은일수}), 추정가격
- 적합도 표시 (높음/중간/낮음) 및 총점
- 출처 플랫폼
- 첨부파일 유무
- 마감 3일 이내 공고는 `[긴급]` 태그 표시
