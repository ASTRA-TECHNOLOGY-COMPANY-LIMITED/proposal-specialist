# 종합 분석 (전체 워크플로우)

기업 문서와 RFP를 기반으로 모든 분석을 순차적으로 수행하라.

$ARGUMENTS 형식: `<RFP경로> [시드경로]`

## 인자 파싱 및 문서 확보

다음 순서로 문서를 확보한다:

1. **$ARGUMENTS에서 확인**: 첫 번째 인자를 RFP 경로, 두 번째 인자를 시드 경로로 사용한다
2. **자동 탐색**: 인자가 없으면 다음을 순차 탐색한다
   - `data/business/` 디렉토리에서 Glob Tool로 RFP/제안요청서 파일 검색
   - `data/seed/` 디렉토리에서 Glob Tool로 기업 시드 문서 검색
   - 파일이 여러 개 발견되면 목록을 보여주고 사용자에게 선택을 요청한다
3. **사용자에게 요청**: 문서를 찾지 못하면 대화형으로 요청한다:
   "분석 대상 문서를 찾지 못했습니다. 다음을 입력해주세요:
   - RFP/제안요청서 파일 경로 (필수)
   - 기업 시드 문서 경로 (선택)"
   - 사용자가 경로를 입력하면 해당 파일로 계속 진행한다

## 실행 계획 확인

분석 파이프라인을 보여주고 실행 범위를 확인한다:

```
## 종합 분석 파이프라인

| # | 단계 | 커맨드 | 설명 |
|---|------|--------|------|
| 1 | 기업 분석 | analyze-company | 기업 역량, 기술 스택, 검색 키워드 도출 |
| 2 | FP 산정 | analyze-fp | 기능점수 산정 및 SW 개발 대가 산출 |
| 3 | 리스크 분석 | analyze-risk | 6대 카테고리 리스크 식별 및 평가 |
| 4 | 사업성 검토 | analyze-feasibility | 수익성/PWin/Bid-No-Bid 종합 검토 |

전체 분석을 실행하시겠습니까?
(Y: 전체 / 번호 지정: 선택 실행 / N: 취소)
```

사용자가 번호를 지정하면 해당 단계만 실행한다 (예: `1 3` → analyze-company + analyze-risk만 실행).

## Phase 1: 기업 분석 (`analyze-company`)

### Step 1: 기업 시드 문서 분석

시드 문서를 doc-analyzer 에이전트로 분석한다:
- 기업 프로필 생성 (업종, 규모, 기술 스택, 인증/자격)
- 주요 수행 실적
- 검색 키워드 도출

### Step 2: 사용자 확인
기업 프로필과 키워드를 보여주고 보완 의견을 받는다.

## Phase 2: FP 산정 (`analyze-fp`)

### Step 3: 기능점수 산정

RFP 문서를 fp-analyzer 에이전트에 위임한다:
- 산정 방법: 간이법 (기본값, 사용자가 정규법 요청 시 변경)
- 기능 요구사항에서 ILF/EIF/EI/EO/EQ 식별
- FP 계산 및 SW 개발 대가 산출

### Step 4: FP 결과 보고
산정 결과 요약을 보여준다:
- 총 기능점수, 예상 대가, 공고 예산 대비 비교

## Phase 3: 리스크 분석 (`analyze-risk`)

### Step 5: 리스크 식별 및 평가

RFP 문서와 기업 시드를 risk-analyzer 에이전트에 위임한다:
- 6대 카테고리 리스크 식별 (자격, 재무, 기술, 수행, 법률, 전략)
- 5x5 확률-영향 매트릭스 스코어링
- Top 5 핵심 리스크 도출

### Step 6: 리스크 결과 보고
핵심 리스크와 대응 전략 요약을 보여준다.

## Phase 4: 사업성 검토 (`analyze-feasibility`)

### Step 7: 종합 사업성 검토

이전 Phase의 결과를 모두 활용하여 feasibility-reviewer 에이전트에 위임한다:
- 이전 단계 결과 컨텍스트 전달: 기업 프로필, FP 산정 결과, 리스크 분석 결과
- Shipley PWin 산출 (6요소 모델)
- 수익성 시뮬레이션 (Best/Expected/Worst)
- Bid/No-Bid 의사결정 매트릭스

### Step 8: 최종 종합 보고

모든 분석 결과를 종합하여 최종 보고서를 출력한다:

```
## 종합 분석 결과

### 1. 기업 프로필 요약
- 핵심 역량: {역량}
- 주요 실적: {실적}

### 2. FP 산정 결과
- 총 기능점수: {FP}점
- 예상 대가: {금액}원
- 공고 예산 대비: {비율}%

|---|--------|------|-----------|
| 1 | {리스크명} | {높음/중간/낮음} | {전략} |
...

### 4. 사업성 검토 결과
- PWin (수주 확률): {%}
- 예상 수익률: {%}
```
