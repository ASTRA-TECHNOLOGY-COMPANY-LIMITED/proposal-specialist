# 기업 문서 분석

$ARGUMENTS 에 지정된 기업 문서를 분석하라.

## 인자 파싱 및 파일 확보

다음 순서로 기업 문서 경로를 확보한다:

1. **$ARGUMENTS에서 확인**: 인자가 있으면 해당 경로를 사용한다
2. **자동 탐색**: 인자가 없으면 다음을 순차 탐색한다
   - `data/seed/` 디렉토리에서 Glob Tool로 기업 관련 문서(`.md`, `.pdf`) 검색
   - 환경 변수 `COMPANY_DOCS_DIR`에 지정된 디렉토리
   - 현재 디렉토리에서 기업 소개서나 기술 문서 검색
3. **사용자에게 요청**: 위 방법으로도 문서를 찾지 못하면 사용자에게 대화형으로 요청한다:
   "분석할 기업 문서를 찾지 못했습니다. 기업 소개서 또는 기술 문서의 경로를 입력해주세요. (파일 또는 디렉토리 경로)"
   - 사용자가 경로를 입력하면 해당 경로로 계속 진행한다
   - 디렉토리를 입력하면 Glob Tool로 하위에서 문서 파일을 자동 식별한다

## 분석 절차

1. 위 절차로 확보한 경로에서 기업 관련 문서를 Read Tool로 읽는다
2. doc-analyzer 에이전트를 호출하여 심층 분석을 수행한다
3. 결과를 정리하여 보고한다
