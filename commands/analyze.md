# 기업 문서 분석

$ARGUMENTS 에 지정된 기업 문서를 분석하라.
인자가 없으면 환경 변수 `COMPANY_DOCS_DIR`에 지정된 디렉토리에서 기업 문서를 찾아라.
`COMPANY_DOCS_DIR`도 설정되지 않았으면 현재 디렉토리에서 기업 소개서나 기술 문서를 찾아라.

## 분석 절차

1. 문서 경로를 결정한다: $ARGUMENTS > COMPANY_DOCS_DIR 환경 변수 > 현재 디렉토리 순
2. 해당 경로에서 기업 관련 문서를 Read Tool로 읽는다
2. doc-analyzer 에이전트를 호출하여 심층 분석을 수행한다
3. 결과를 정리하여 보고한다
