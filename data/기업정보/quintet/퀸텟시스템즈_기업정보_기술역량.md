## 2. 핵심 사업 영역

### 2.1 CRM 솔루션 — iCIGNAL

퀸텟시스템즈의 대표 CRM 솔루션으로, 국내 CRM 시장 1위를 차지하고 있으며 **가트너(Gartner) CRM 벤더가이드에 2년 연속 등재**된 글로벌 수준의 솔루션입니다.

| 항목 | 내용 |
|------|------|
| 솔루션명 | iCIGNAL (아이시그널) |
| 유형 | 구축형 CRM (엔터프라이즈) + SaaS형 (중견·중소기업) |
| 핵심 기능 | 고객360 뷰, 멤버십 정책 수립, 통합 로열티 프로그램, 마케팅 자동화 |
| 인증 | 가트너 CRM 벤더가이드 등재 (2017, 2018) |

#### iCIGNAL CRM 기능 영역

| 기능 영역 | 주요 기능 |
|---------|---------|
| **멤버십 (Loyalty)** | 멤버십 정책 관리, 포인트·프로모션·마케팅 온/오프라인 통합 |
| **마케팅 (Marketing)** | 고객 세분화, 캠페인 관리, 실시간 마케팅 자동화, BI/분석 |
| **세일즈 (Sales)** | 영업기회 관리, 조직관리, 성과 플로우 관리 |
| **서비스 (Service)** | 콜센터 통합, 상담 관리, 고객 서비스 자동화 |
| **데이터분석** | AI 기반 분석 지표 추천, 드래그앤드롭 리포트 생성, DW 연계 |
| **옴니채널** | 멀티채널 통합, O2O 서비스, 디지털 마케팅 |

### 2.2 로우코드 개발 플랫폼 — CALS Platz

**국내 최초 SaaS 로우코드 개발 플랫폼**으로, AWS 기반 클라우드 환경에서 기업용 애플리케이션을 빠르게 개발할 수 있는 플랫폼입니다.

| 항목 | 내용 |
|------|------|
| 플랫폼명 | CALS Platz (Cloud At the Light Speed) |
| 출시연도 | 2020년 |
| 인프라 | AWS (Amazon Web Services) 기반 |
| 파트너십 | AWS ISV Accelerated Partnership |
| 아키텍처 | MSA (Microservice Architecture), 멀티테넌트, SaaS |

#### CALS 핵심 기능

| 기능 | 설명 |
|------|------|
| **런타임 소스코드 제너레이션 엔진** | 코드 자동 생성으로 개발 속도 극대화 |
| **애플리케이션 프로비저닝** | 클라우드 환경에서 앱 자동 배포 및 관리 |
| **테넌트 관리** | 멀티테넌트 환경에서 비즈니스 프로세스 최적화 |
| **서비스 오케스트레이션** | 스케줄링·이벤트 트리거 기반 서비스 자동화 |
| **빌드 파이프라인** | CI/CD 자동화, 기존 배포 시간 40% 단축 |
| **Open API / REST API 연동** | 외부 서비스, AWS Lambda 등 다양한 시스템 연계 |

#### CALS 생산성 지표

| 지표 | 수치 |
|------|------|
| 개발 생산성 향상 | 자체 개발 대비 **10~20배** |
| Cloud Migration 비용 절감 | 최대 **40%** |
| 유지보수 비용 절감 | **20~30%** |
| 비전문가 학습 기간 | 약 **2주** 내 솔루션 구현 가능 |
| 배포 시간 단축 | 빌드 파이프라인 활용 시 **40%** 단축 |
