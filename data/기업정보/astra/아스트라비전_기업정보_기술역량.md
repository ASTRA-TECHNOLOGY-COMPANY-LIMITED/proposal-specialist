## 6. 자체 AI 플랫폼

### 6.1 APEX (AI Platform for Export) — 수출 전주기 AI 통합 플랫폼

APEX는 자체 개발한 수출 전주기 AI 통합 플랫폼으로, 콘텐츠 제작부터 전략 수립, 바이어 매칭까지 원스톱으로 지원합니다. 멀티테넌트 아키텍처로 **250개 기업 동시 지원** 가능.

| 모듈 | 주요 기능 | 기술 스펙 |
|------|---------|---------|
| **수출전략 수립** | 7단계 프로세스, 429개 체크리스트, 6명 AI 페르소나 분석 | LangGraph 에이전트 워크플로우 |
| **디지털 콘텐츠 제작** | 다국어 자동 생성, 플랫폼별 규격 변환, 문화적 맥락 반영 | GPT-4, 멀티모달 AI |

## 7. 핵심 기술 역량

### 7.1 AI 에이전트 개발

| 에이전트 패턴 | 설명 | 적용 사례 |
|------------|------|---------|
| ReAct (Reasoning + Acting) | 생각-행동-관찰을 반복하며 추론 | 복잡한 문의 응대 챗봇 |
| Plan-Execute-Replan | 계획 수립 후 단계별 실행 및 재계획 | 업무 자동화 에이전트 |
| Multi-Agent Orchestration | 여러 전문 에이전트의 협업 조율 | B2B 세일즈 응대 시스템 |
| Tool-Calling Agent | 외부 도구와 API 연동 실행 | 데이터 조회/분석 에이전트 |
| Iterative Retrieval Agent | 반복적 검색 및 정보 축적 | 문서 분석 에이전트 |

- **상태 기반 워크플로우 관리**: LangGraph StateGraph 활용
- **조건부 라우팅**: 사용자 의도에 따른 동적 워크플로우 분기
- **Human-in-the-Loop**: 에스컬레이션 설계
- **체크포인트 관리**: 대화 중단 시 상태 저장/복원

### 7.2 RAG 시스템

8단계 RAG 파이프라인: 문서 파싱 → Semantic Chunking → 임베딩 생성 → 벡터 인덱싱 → 유사도 검색 → Re-ranking → 응답 생성 → Grounding 검증

| 고급 검색 기술 | 설명 |
|------------|------|
| Hybrid Search | 벡터 검색 + 키워드 검색 결합 |
| Multi-hop Retrieval | 여러 단계 정보 수집 |
| Query Decomposition | 복잡한 질문 하위 분해 |
| Adaptive Retrieval | 정보 충분성 기반 반복 검색 |
| MMR | 검색 결과 다양성 확보 |

**멀티테넌트 RAG 아키텍처**: 250개+ 기업 동시 서비스, 기업별 독립 벡터 인덱스·대화 이력·설정 관리

### 7.3 바이어-셀러 매칭 시스템

- **Vector DB + Knowledge Graph 하이브리드 매칭** (Precision@10: 0.80~0.90)
- Neo4j 기반 지식 그래프 (바이어, 셀러, 제품, HS Code, 인증, 국가 엔티티)
- 메타데이터 필터링 + 벡터 검색 결합 (10,000건 → 500건 사전 필터링, 검색 속도 20배 향상)
- Cross-encoder 리랭킹 + Graph 관계 기반 부스팅

### 7.4 대용량 데이터 처리

| 역량 | 수치 |
|------|------|
| Excel 처리 | **10만행+** (Streaming, Semantic Chunking) |
| 메시지 처리 | **일 100만건+** (RabbitMQ 기반 분산 시스템) |
| 동시 처리 | 무제한 (Consumer Pool 수평 확장) |
| 평균 응답 시간 | 5초 이내 |
| 가용성 | 99.9% 이상 |

### 7.5 B2B 세일즈 챗봇

- 24시간 해외 바이어 자동 응대
- 7개 카테고리 의도 분류 (제품문의, 가격, MOQ, 배송, 인증 등)
- **4개국어 지원** (한국어, 영어, 중국어, 베트남어)
- 바이어 정보 자동 수집·분석
- Grounding Verification으로 Hallucination 방지
