### 2.1 베트남 정산 관리 서비스 (VLE — Vietnam Liquidation Engine)

VLE는 아스트라테크의 핵심 자체 솔루션으로, 베트남 진출 제조기업의 수입관세 정산 업무를 자동화하는 시스템입니다.

| 항목 | 내용 |
|------|------|
| 솔루션명 | VLE (Vietnam Liquidation Engine) 2.0 |
| 서비스 URL | https://vle.rudasoft.com |
| 최초 개발 | 2021년 1월 (루다소프트 자체 개발) |
| 특허 | 엑셀 문서 인공지능 변환 시스템 (제10-2563961호) |
| 구축 실적 | **11개사 이상** |

**2011년부터 구축한 국가 핵심 시스템**으로, 아스트라테크(구 루다소프트)가 최초 개발사입니다.

| 항목 | 내용 |
|------|------|
| 시스템명 | FTA Korea Plus (표준형, 간편형, Plus, FTA Agri) |
| 발주 기관 | 산업통상자원부 / (주)한국무역정보통신(KTNET) |
| 최초 개발 | 2011년 (FTA 원산지 관리 솔루션) |
| 연속 수행 | **2018~2025 연속 8년** 종합컨설팅 + 유지보수 |
| 기술 스택 | Java, Spring Boot, Angular, REST API, JPA, CI/CD |
| 연계 기관 | 관세청(UNI-PASS), 대한상공회의소, 원산지증명센터 |

| 유형 | 설명 | 대상 기업 |
|------|------|---------|

| 오토링크(서비스형) | KTNET 공용 FTA 서비스 사용 | 별도 서버 구축 어려운 기업 |
| 플러그인(Data Mapper) | 엑셀 업로드 방식의 인터페이스 | 소규모 기업 |

#### 주요 고도화 기능 (2025년)

- 원산지확인서 거래처별 주요 품목/자재 관리 기능
- 원산지확인서 수취대상 자동 지정 기능
- 신규 FTA 협정 기관발급 연계 (한-필리핀, 한-UAE, 한-에콰도르, 한-GCC)
- 원산지 판정 후 결정기준 충족 관련 기능 개선
- 원산지 판정 속도 개선 (BOM 전개 프로세스 최적화)
- 협력사 지원 기능 개선 (협력기업 포털 대시보드)
- 고객지원·교육 관련 게시판, 관리자 상담 내역 관리

## 4. 기술 역량

### 4.1 엔터프라이즈 시스템 개발 (Java/Spring Boot 기반)

아스트라테크의 기술적 근간은 **엔터프라이즈급 Java/Spring Boot 시스템 개발** 역량입니다. FTA Korea 시스템을 최초 개발하고 8년간 연속 운영·고도화한 경험을 보유하고 있습니다.

| 기술 영역 | 적용 기술 | 비고 |
|---------|---------|------|
| **플랫폼** | WAS, WEB, DBMS, J2EE, Spring Boot | 대규모 엔터프라이즈 시스템 |
| **요소기술** | Java, Angular, CSS3, HTML5, XHTML 1.0 | 프론트엔드 + 백엔드 풀스택 |
| **인터페이스** | XML, REST API, CSV, 메타데이터 | ERP/관세청/기관 연계 |
| **기반기술** | 표준프레임워크, REST API, JPA, CI/CD | 현대적 개발 방법론 |
| **보안** | SSL, 시큐어코딩, 관리자권한관리, SW개발보안 | 공공시스템 보안 기준 준수 |

### 4.2 AI/ML 기술 역량 (아스트라비전 합산)

아스트라비전 합류 이후 AI 기술 역량이 대폭 강화되었습니다.

| 기술 | 용도 | 적용 사례 |

| OpenAI GPT-4 / Claude | 응답 생성 엔진 | B2B 챗봇, 콘텐츠 자동 생성 |
| FAISS / Qdrant | 벡터 유사도 검색 | RAG 파이프라인 |
| Neo4j | 지식 그래프 | 바이어-셀러 매칭 |
| Sentence Transformers | 임베딩 모델 | 시멘틱 검색 |

### 4.4 도메인 특화 기술

#### 원산지관리 시스템 기술
- FTA 협정 정보 API, 원산지 결정기준 API, 코드정보 API
- 원산지확인서 관리/발급 연계 (관세청 UNI-PASS, 대한상공회의소)
- 원산지 판정 엔진 (BOM 전개, 결정기준 자동 충족 판정)
- Data Mapper를 활용한 엑셀 업로드 방식 자동데이터 등록
- 오토링크 방식 ERP 연계 (설치형/서비스형)

#### 베트남 무역 자동화 기술
- 베트남 통관시스템 VNACCS 연계
- VLE 정산 리스크 분석 및 보고
- 수입관세 재고 현황 자동 관리
- 기업 ERP ↔ 베트남 세관 시스템 자동 연동

#### AI 에이전트 기술
- ReAct, Plan-Execute-Replan, Multi-Agent Orchestration 패턴
- 멀티테넌트 RAG 아키텍처 (250개+ 기업 동시 서비스)
- Hybrid Search (Vector DB + Knowledge Graph)
- Grounding Verification (Hallucination 방지)
- 4개국어 자동 감지 및 응대 (한/영/중/베트남어)

- **20명 규모의 현지 개발·운영 인력** 보유
- VLE 11개사+ 구축으로 검증된 현지 사업 수행 능력
- 베트남 통관시스템(VNACCS) 연계 기술 보유

### 5.3 무역 도메인 전문성
- 관세사 자격 보유 대표이사 (15년 무역 실무 경력)
- 원산지관리사, 물류관리사, 구매자재관리사 보유 인력

- FTA 협정, HS코드, 관세율, 원산지 결정기준 등 무역 법규 전문 지식

### 5.4 AI + 무역 융합 역량
- 전통적 엔터프라이즈 시스템(Java/Spring Boot) 기반 위에
- 최신 AI 기술(LangGraph, RAG, Knowledge Graph)을 접목
- "AI 생성 + 전문가 검수" 하이브리드 운영 모델
- APEX 플랫폼으로 수출 전주기 AI 자동화 실현

### 5.5 공공사업 수행 경험

- ODA 사업 경험 (베트남, 에티오피아, 우즈베키스탄, 페루 등)
